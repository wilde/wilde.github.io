一、集成式开发环境

集成开发环境（IDE，Integrated Development Environment ）是用于提供程序开发环境的应用程序， 一般包括代码编辑器、编译器、调试器和图形用户界面等工具。集成了代码编写功能、分析功能、编译功能、 调试功能等一体化的开发软件服务套。所有具备这一特性的软件或者软件套（组）都可以叫集成开发环境。 （转自百度百科）

1. 在本课程中，我们会学习使用Web设计领域中的标志性工具Dreamweaver。 此链接是Dreamweaver的官方主页，在这里可以找到Dreamweaver最新版本的软件下载和教程。

2. 另一个推荐的IDE是WebStorm。与更侧重于设计的Dreamweaver不同，WebStorm对Web开发提供了更多的支持。由于开发 工作的复杂性，使得此款软件具有一定的学习曲线，但是花些时间掌握WebStorm与其带来的生产效率提供 是成正比的。在这里，可以发现软件的 下载和教程，并且JetBrains还提供了对教育的免费政策。

3. HBuilder是一款国产的免费开发工具，它兼具了 IDE和编辑器的特长。其特别之处在于针对中国的互联网环境，增加了许多特色的功能，如对开发小程序的支持。

二、编辑器

编辑器也被称作文本编辑器，与IDE不同之处在于许多功能并没有集成到软件中，许多的编辑功能需要手动编辑，甚至是 通过命令行的方式来实现。但相较于Windows操作系统自带的文本编辑器“记事本”来说，其具有的功能更多也更强大。 对于刚接触编程的人来说，使用编辑器可以更好地熟悉代码的构成和运行；对于有一定经验的程序员来说，也更愿意使用 运行速度更快且功能不逊于IDE的编辑器来完成工作。

1. Visual Studio Code，这是由微软出品的免费并且开源 的编辑器。它虽然是Visual Studio的简化版，但凭借其社区不断的更新和大量的扩展，使其拥有了大量的使用者。并且它 还提供了网页版本，这使得使用Pad的用户也可以通过打开网页，登录自己的账号3 就可以使用并同步自己的工作了。

2. Sublime Text虽然是一款商用软件，但它并非强制使用者付费——这一点 与著名的文件管理器Total Commander类似。并且它也以运行速 度快，以及拥有众多的开发者和插件而闻名。

四、代码托管

想让你的网站展示在互联网上吗，那么你需要一个网站空间来存放页面、图片等文件。鉴于本课程主要涉及到的是静态 页面，与复杂的需要存储数据的动态网站相比，我们可以使用一些互联网服务商的静态页面托管服务，只需要注册一个账号 上传需要展示的文件即可。

1. Github是当今最大的代码托管网站之一，它的学习和使用难度较大，但其 为你带来的未来回报是值得花些时间的。

2. Github Pages严格意义上来说是GitHub一个额外的功能，可以对 仓库中的页面实时地展示出来，

3. 鉴于代码托管的安全和稳定性，国产的gitee是除了github之外同步使用 代码托管网站。gitee还提供了高校版，提供了一些适用于教育环境以及本地化的应用。

4. cloudflare主要是提供互联网安全和云服务的一家公司，注册后在仪表板中 可以找到“Pages”，其提供与Github Pages类似的功能。
